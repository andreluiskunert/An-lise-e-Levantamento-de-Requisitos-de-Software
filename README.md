# An-lise-e-Levantamento-de-Requisitos-de-Software
O curso “Análise e Levantamento de Requisitos de Software” ensina técnicas de coleta, análise, documentação, validação e gestão de requisitos em projetos de software. Aborda UML, prototipação, métodos ágeis, comunicação com stakeholders, estudos de caso e ferramentas modernas, preparando alunos e profissionais para atuar na análise de requisitos.
