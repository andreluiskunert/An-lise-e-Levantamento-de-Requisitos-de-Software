# 📘 Análise e Levantamento de Requisitos de Software

## 📖 Descrição

O curso **Análise e Levantamento de Requisitos de Software** foi desenvolvido para estudantes e profissionais de TI que desejam aprender técnicas modernas de análise de requisitos em projetos de software. O conteúdo aborda desde conceitos fundamentais até práticas avançadas utilizadas no mercado.

Durante o curso, o aluno aprende a coletar, analisar, documentar, validar e gerenciar requisitos de software utilizando metodologias, ferramentas e técnicas amplamente aplicadas na área de desenvolvimento.

---

## 🚀 Conteúdos Abordados

- Introdução à análise de requisitos
- Elicitação de requisitos
- Documentação e especificação de requisitos
- Verificação e validação de requisitos
- Modelagem UML
- Prototipação
- Gestão de requisitos
- Comunicação com stakeholders
- Métodos ágeis e User Stories
- Estudos de caso e exercícios práticos
- Ferramentas modernas de análise de requisitos
- Tendências atuais como IA e análise preditiva

---

## 🛠️ O que você aprenderá

- Técnicas de levantamento de requisitos
- Entrevistas, brainstorming e workshops
- Diagramas UML e modelagem de dados
- Escrita clara e sem ambiguidades
- Gestão de mudanças e rastreabilidade
- Prototipação rápida
- Comunicação e negociação com stakeholders
- Integração dos requisitos ao desenvolvimento de software
- Análise orientada a objetos
- Uso de ferramentas colaborativas
- Todos os 14 diagramas UML

---
