Primeiro de todos os requisitos funcionais.

Como já mencionado anteriormente, os requisitos funcionais descrevem as funcionalidades e tarefas específicas

que um software deve ser capaz de realizar.

Eles definem as ações que o sistema deve executar em resposta a determinadas entradas, ou seja, a

determinadas ações efetuadas pelos usuários.
